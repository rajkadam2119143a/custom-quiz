== Description ==

The Custom Quiz System is a powerful WordPress plugin that provides a complete assessment solution with the following features:

**Key Features:**
- Excel/CSV import for bulk question management
- Proportional question distribution by category
- 2-hour timed quiz with auto-submission
- Real-time progress saving
- Detailed category-wise result reporting
- Modern, responsive UI
- User assignment system
- URL-based state management

**Question Types Supported:**
- Single choice questions
- Multiple choice questions
- Text input questions
- True/False questions

**Advanced Features:**
- Auto-save functionality
- Progress tracking
- Category-wise performance analysis
- Detailed question review with explanations
- Print-friendly results
- User session management

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/custom-quiz/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to 'Custom Quiz' in the admin menu to configure settings
4. Import questions using the Excel/CSV import feature
5. Use the shortcode `[quiz_assessment]` to display the quiz on any page

== Frequently Asked Questions ==

= How do I import questions? =

1. Go to Custom Quiz > Import Questions
2. Download the CSV template
3. Fill in your questions following the template format
4. Upload the CSV file
5. Click "Import Questions"

= Can I customize the quiz settings? =

Yes! Go to Custom Quiz > Settings to configure:
- Time limit
- Number of questions per quiz
- Question distribution method
- Result display options

= How do I display the quiz on my site? =

Use the shortcode `[quiz_assessment]` on any page or post where you want the quiz to appear.

= Can users retake the quiz? =

This can be configured in the settings. By default, users can only take the quiz once.

= How are results calculated? =

Results are calculated based on correct answers and include:
- Overall percentage score
- Category-wise performance
- Detailed question review
- Time taken
